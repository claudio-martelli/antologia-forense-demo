# Assets — Workflow di generazione immagini

Materiale relativo al processo di generazione delle illustrazioni del gioco (carte, schede, tabellone), in tre fasi:

1. **Sketch** — Pessima bozza disegnata a mano su tablet, suddivisione in colori per oggetto/scopo in funzione leggibilità modello IA
2. **Nanobanana** — generazione dell'immagine tramite prompt strutturato, con estetica noir/sepia coerente con l'ambientazione anni '30
3. **Cardmaker** — impaginazione finale in stile Magic: The Gathering

- **`bozze-generazioni-nanobanana.pdf`** — raccolta delle bozze e degli esiti del workflow descritto sopra.

Per il dettaglio testuale del workflow, vedi il paragrafo dedicato nel report tecnico ([`../docs/report-tecnico-v0.1.pdf`](../docs/report-tecnico-v0.1.pdf)).
