# Partite di esempio

Questa cartella raccoglie le trascrizioni di sessioni di gioco reali, generate facendo interagire un LLM (configurato come Voce Narrante secondo `prompt/prompt-master.txt` + `prompt/cronogramma.md`) con dei giocatori.

## Nota

- Il file **`-raw.txt`** contiene la trascrizione integrale e non modificata, incluso il prompt di sistema iniziale e le note di ragionamento interno del modello (utile per analisi tecniche o debug del prompt).
- Il file **`.md`** è una versione editata e leggibile, pensata per essere letta come "resoconto di partita": sono state rimosse le parti di ragionamento interno del modello, mantenendo dialoghi, narrazione e meccaniche di gioco (uso delle carte, ricompense, deduzioni).
- Il link alla conversazione originale può scadere o essere reso privato da DeepSeek: i file di questa cartella ne sono la copia permanente di riferimento.

Per aggiungere una nuova partita di esempio, segui lo stesso schema: carica il file grezzo, producine una versione leggibile, e aggiorna la tabella qui sopra.
