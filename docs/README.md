# Documentazione di progetto

Questa cartella raccoglie tutta la documentazione "di redazione" del progetto: regole, relazioni d'esame e report.

| File | Contenuto |
|---|---|
| [`manuale-di-gioco.md`](manuale-di-gioco.md) | Manuale di gioco completo (v0.3): introduzione, trama, le 4 carte, modalità di gioco, codici tutor, schede dei 10 personaggi. È il documento da leggere per imparare a giocare. |
| [`meccaniche-di-gioco.md`](meccaniche-di-gioco.md) | Relazione d'esame sulle meccaniche scelte (carte a risorse limitate, narrazione LLM, ruolo del cronogramma). Conversione in Markdown di `ANTOLOGIA_FORENSE_Meccaniche_di_Gioco.docx`. |
| [`concettualizzazione-prototipo.md`](concettualizzazione-prototipo.md) | Relazione d'esame sulla fase di ideazione e concettualizzazione (tema, obiettivi, target, loop di gioco, arbitraggio, valutazione). Conversione in Markdown di `ANTOLOGIA_FORENSE_concettualizzazione_del_prototipo_di_gioco.docx`. |
| [`report-tecnico-v0.1.pdf`](report-tecnico-v0.1.pdf) | Report tecnico/funzionale/didattico che sintetizza tutti i materiali del progetto, con struttura suggerita per una presentazione PowerPoint. |
| [`presentazione-progetto.pdf`](presentazione-progetto.pdf) | Presentazione generale del progetto. |

## Ordine di lettura consigliato

1. `concettualizzazione-prototipo.md` — per capire **perché** il gioco è progettato così
2. `meccaniche-di-gioco.md` — per capire **come** funzionano le meccaniche nel dettaglio
3. `manuale-di-gioco.md` — il **regolamento operativo**, da usare per giocare davvero
4. `report-tecnico-v0.1.pdf` / `presentazione-progetto.pdf` — sintesi pre-finale e proposta materiale di presentazione
